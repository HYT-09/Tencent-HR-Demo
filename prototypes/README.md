# 腾讯校园招聘 App 高保真原型

## 项目概述

本项目为腾讯校园招聘移动应用（用户端+HR端）及运营后台管理系统的高保真原型设计，基于 iPhone 15 Pro 设计规范，使用 HTML + CSS 构建，可直接用于开发参考。

## 文件结构

```
prototypes/
├── index.html              # 主入口页面（展示所有原型）
├── styles/
│   └── app.css            # 全局样式与组件库
├── user/                   # 用户端页面
│   ├── home.html          # 首页
│   ├── resume_upload.html # 简历上传
│   ├── matching_result.html # 匹配结果
│   ├── apply.html         # 投递管理
│   ├── job_progress.html  # 求职进度看板
│   └── user_center.html   # 个人中心
├── hr/                     # HR端页面
│   ├── hr_dashboard.html  # HR工作台
│   ├── hr_candidate_list.html # 候选人列表
│   └── hr_reject_pool.html # 候选人回捞
└── admin/                  # 运营后台页面
    └── admin_dashboard.html # 运营后台概览
```

## 功能模块

### 用户端（小程序/APP）

1. **首页 (home.html)**
   - 核心CTA「上传简历 智能匹配岗位」按钮
   - 数据统计展示（在招岗位、今日新增、平均匹配度）
   - 最新公告轮播
   - 热门在招岗位推荐

2. **简历上传 (resume_upload.html)**
   - 拍照上传 / 文件上传两种方式
   - 上传进度展示（图片识别 → AI解析中）
   - 最近上传简历管理
   - 上传小贴士

3. **匹配结果 (matching_result.html)**
   - 简历概览（学历、专业、技能）
   - 匹配分数可视化（各维度权重得分）
   - 2-3个最优岗位推荐（S/A/B级）
   - 详细匹配理由
   - 一键投递按钮

4. **求职进度看板 (job_progress.html)**
   - 全局进度时间轴
   - 三大状态分类（已投递/已获邀/已拒绝）
   - 面试详情（时间、地点、形式）
   - 面试确认/拒绝操作

5. **投递管理 (apply.html)**
   - 全部投递列表
   - 状态筛选
   - 投递详情与进度追踪
   - 拒绝岗位AI回捞入口

6. **个人中心 (user_center.html)**
   - 用户信息与简历管理
   - 快捷操作入口
   - 求职数据统计
   - 设置与帮助

### HR端

1. **HR工作台 (hr_dashboard.html)**
   - 今日待处理任务
   - 高匹配候选人推送（S/A级）
   - 投递数据统计
   - 最近操作记录

2. **候选人列表 (hr_candidate_list.html)**
   - 全部候选人管理
   - 岗位/状态筛选
   - 匹配度排序
   - 批量操作（发起邀约/标记拒绝）

3. **候选人回捞 (hr_reject_pool.html)**
   - 已拒绝候选人池
   - AI二次匹配建议
   - 回捞状态追踪
   - 批量回捞操作

### 运营后台

1. **数据概览 (admin_dashboard.html)**
   - 岗位数据统计（总岗位/有效/新增/失效）
   - 投递趋势图表
   - 简历匹配等级分布
   - 最新入库岗位列表
   - 运维操作入口

## 设计规范

### 设备适配
- iPhone 15 Pro: 393 × 852px
- 圆角: 47px
- 状态栏高度: 54px
- 底部Tab Bar高度: 83px

### 色彩系统
```
主色（腾讯红）: #E64340
辅助蓝: #2A5CFF
成功绿: #34C759
警告橙: #FF9500
错误红: #FF3B30
中性灰: #6B7280
背景色: #F8F9FA
```

### 匹配等级
```
S级: ≥90%   深红色标签
A级: 80-89% 橙色标签
B级: 70-79% 绿色标签
C级: <70%   灰色标签
```

### 匹配权重规则
```
学历匹配: 30%
专业匹配: 20%
技能匹配: 30%
经历匹配: 20%
总计: 100%
```

## 使用方式

1. 直接在浏览器中打开 `index.html`
2. 使用页面导航快速跳转至各模块
3. 原型以实际设备尺寸展示，可滚动查看完整内容
